# 🌌 Aura 2026: Interactive Builder

> A plataforma elite de transmutação de código e design, potencializada por IA multilinear.

Aura 2026 (Interactive Builder) é um ecossistema de alta performance para criação de materiais digitais. Utilizando uma arquitetura de "Design Archetypes", a plataforma permite converter estratégias de branding em interfaces funcionais e esteticamente superiores em segundos.

---

## ⚡ Início Rápido

### Pré-requisitos
- Node.js 18+
- Conta no Supabase
- Chave de API (Gemini, OpenAI, Claude ou Groq)

### Instalação
1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure o arquivo `.env.local` (veja `.env.example`):
   ```env
   VITE_SUPABASE_URL=sua_url
   VITE_SUPABASE_ANON_KEY=sua_chave
   ```
4. Inicie o motor:
   ```bash
   npm run dev
   ```

---

## 💎 Funcionalidades Elite

- **Transmutação Multilinear**: Gere código HTML/CSS premium usando Gemini, GPT-4, Claude 3.5 ou Llama 3.
- **Arquitetura de Branding**: Crie e gerencie presets de marca (Cores, Fontes, Tom de Voz) independentes.
- **Biblioteca de Materiais**: Repositório centralizado com versionamento por estilo e exportação massiva via **ZIP**.
- **Interface Aura Dinâmica**: Personalize a cor de destaque da plataforma sem afetar o branding dos clientes.
- **Visualização em Tempo Real**: Preview instantâneo com suporte a metadados SEO e tags de estilo.
